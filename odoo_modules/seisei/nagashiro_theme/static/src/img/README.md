# Logo and Favicon Images

Place your custom logo and favicon files here:

- **logo.png**: Main logo (recommended size: 200x60px or similar)
- **favicon.ico**: Browser favicon (recommended size: 32x32px or 16x16px)

## Default Behavior

If no custom images are provided, Odoo will use its default logo and favicon.

## Custom Images

To use custom images:

1. Place your logo file as `logo.png` in this directory
2. Place your favicon file as `favicon.ico` in this directory
3. Restart Odoo
4. The theme will automatically use these images

## Image Requirements

- **Logo**: PNG format, transparent background recommended
- **Favicon**: ICO format, 32x32px or 16x16px
