Use Cases

Multi-Station Enterprise

Unified printer management across multiple work stations with independent print policies for each location, enabling centralized management and monitoring across regions.

Department-Level Management

Different departments using different printers, automatically routing reports to corresponding devices through mapping rules, improving work efficiency.

Printer Failover

Automatic task transfer to backup printers when primary printer fails, ensuring business continuity and reducing equipment failure impact.

Print Job Queue

Manage print job queues with priority settings, optimally allocate print resources and optimize printing device utilization.

Remote Print Service

Integration with external print services supporting remote printing scenarios, enabling seamless connection between cloud and local printing devices.

Specialized Report Printing

Configure dedicated printers for specific reports such as receipt printers, label printers, meeting different business scenario printing needs.

Quick Start Guide

1

Install Module

Install YLHC Print Manager module from Odoo App Store and update the application list

2

Configure Service Connection

Configure print service connection information in settings, including WebSocket address and authentication parameters

3

Sync Printer Information

Click sync button to retrieve available printer list and status information from print service

4

Setup Mapping Rules

Configure printer mapping rules for different reports, set print policies and priorities

5

Start Using

When printing reports, the system will automatically select appropriate printers based on configured rules to execute print tasks
